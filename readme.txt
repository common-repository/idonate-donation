== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `idonate-donation` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to iDonate and add API key which provided by administrator and get shortcode.
